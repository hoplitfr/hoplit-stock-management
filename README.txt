== Description ==
Hoplit Stock Management is a simple plugin to manage product stock on your WordPress site. It allows you to track items, their quantity, and price.

== Installation ==
1. Download the plugin from the Plugins section in your WordPress dashboard or download it from the WordPress.org repository.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. You can now manage your stock by accessing the "Hoplit Stock Management" tab in the admin menu.

== Usage ==
Once the plugin is activated, go to the "Hoplit Stock Management" tab in the admin menu. You can add, edit, delete, and view items in your stock.

To display the stock table on a page or post, use the shortcode `[wp_stock_management_table]`.

== FAQ ==
= How can I display the stock table on a page or post? =
Use the `[wp_stock_management_table]` shortcode in the content of the page or post where you want to display the stock table.
